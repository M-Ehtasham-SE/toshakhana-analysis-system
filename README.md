# Toshakhana Analysis System

A Probability & Statistics based data analysis system developed to analyze Toshakhana gift records using statistical methods, probability theory, data visualization, and machine learning techniques.

---

## Project Overview

The Toshakhana Analysis System is a Flask-based web application that performs statistical and probabilistic analysis on Toshakhana dataset records.

The project explores:

- Gift retention trends
- Probability distributions
- Statistical summaries
- Conditional probabilities
- Regression analysis
- Predictive machine learning models

This project was developed as part of the Probability & Statistics course.

---

## Features

- Descriptive Statistical Analysis
- Probability Distribution Analysis
- Conditional Probability Computation
- Histogram & Data Visualization
- Linear Regression Analysis
- Logistic Regression Predictions
- Random Forest Classification
- Data Cleaning & Preprocessing
- Interactive Flask Dashboard
